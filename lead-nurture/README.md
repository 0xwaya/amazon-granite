# Urbanstone Lead Nurture Funnel

Purpose: convert sourced leads into booked consultations with a consistent email + SMS cadence.

Status: **draft templates** (no sending automation wired yet).

## Recommended cadence

- Day 0: Immediate thank-you + quick qualifying question
- Day 1: Portfolio + social proof + booking CTA
- Day 3: Material guidance + budget guardrails
- Day 6: Case study + timeline expectations
- Day 10: Last-call follow-up + offer to hold slot
- Day 14: Re-engagement + seasonal promo

## Usage notes

- Use the templates in `templates.json` to generate personalized messages.
- Customize by project type (kitchen, bath, outdoor), material interest, and timeframe.
- Send only after explicit approval.
